## 🎯 **How to Use:**

### **Send HTML Email (Default):**

```
1. Dashboard → "📤 Send Message"
2. Click "📧 Send Email" section
3. Select user
4. Enter subject
5. Enter message
6. ✅ "Use HTML Template" is CHECKED (default!)
7. Click "Send Email"
8. Enter OTP
9. ✅ Beautiful HTML email sent!
```

**User receives:**
- Professional branded email
- Logo and colors
- Styled formatting
- Footer with links

---

### **Send Plain Text Email:**

```
1. Dashboard → "📤 Send Message"
2. Click "📧 Send Email" section
3. Select user
4. Enter subject
5. Enter message
6. ❌ UNCHECK "Use HTML Template"
7. Click "Send Email"
8. Enter OTP
9. ✅ Plain text email sent!
```

**User receives:**
- Simple text-only email
- No formatting
- No branding

## 🔧 **Technical Details:**

### **HTML Template Includes:**

```html
<!DOCTYPE html>
<html>
  <head>
    <style>
      /* Purple gradient header */
      /* Responsive design */
      /* Brand colors */
    </style>
  </head>
  <body>
    <!-- Header with logo -->
    <!-- Your message -->
    <!-- Footer with links -->
  </body>
</html>
```

### **Headers Set:**

**HTML Email:**
```
From: HostingAura <noreply@hostingaura.com>
Reply-To: support@hostingaura.com
MIME-Version: 1.0
Content-Type: text/html; charset=UTF-8
```

**Plain Text:**
```
From: noreply@hostingaura.com
Reply-To: support@hostingaura.com
Content-Type: text/plain; charset=UTF-8
```
